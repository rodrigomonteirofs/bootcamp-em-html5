Conhecendo a estrutura básica do HTML.

HTML ou HyperText Markup Language é uma linguagem de marcação utilizada na construção de páginas web client-side.

A linguagem possui uma estrutura simples onde os comandos são chamados de tags e escritos entre o sinal de < e um sinal de > para sua abertura e para seu fechamento deve-se inserir uma barra logo após o sinal de </> com base na seguinte informação veja abaixo um exemplo:

<!DOCTYPE html>

            <html>

                <head>
                    <title>Título da página</title>
               </head>
            
               <body> </body>

            </html>

O que são tags e qual suas funções?

As tags são usadas para informar ao navegador a estrutura do site ou seja: quando se escreve um código em HTML, as tags serão interpretadas pelo navegador, produzindo assim a estrutura e o conteúdo visual da página. Abaixo serão listadas algumas tags e suas respectivas funções:

  1. <!DOCTYPE html>
    A tag !DOCTYPE informa ao navegador a versão do HTML que está sendo utilizada no documento, vale lembrar que esta tag é uma das excessões do HTML que não precisam de uma outra tag de fechamento.
  2. <html> </html>
    Ela vai carregar todos os elementos do documento.
  3. <title> </title>
    Essa tag delimita o cabeçalho do documento.
